# KI-ARS Frontend

This directory contains the Vue.js single-page application used to display lecture feedback, analytics and course information.

---

## Quick Start

1. Navigate into the frontend directory:

       cd /KI-ARS_Zero-Shot/frontend

2. Install all required dependencies:

       npm install

3. Start the development server:

       npm run dev

The application will be available at:

    http://localhost:5173

The backend must be running on `http://localhost:8080` for the application to function correctly.
